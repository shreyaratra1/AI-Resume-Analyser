# AI-Resume-Analyser
AI-based resume analyzer using ML and NLP
📌 Overview
AI Resume Analyzer is a machine learning-based web application that analyzes resumes and predicts suitable job roles. It also calculates a match score between the resume and job description to help users understand their profile alignment.

🎯 Objective
To build an intelligent system that automates resume screening and provides job role prediction along with similarity matching.

🚀 Features
 Upload or paste resume text
 Predict suitable job role
 Calculate resume-job match score
 NLP-based text processing
 Interactive UI using Streamlit

🛠️ Tech Stack
Python
Pandas
NumPy
Scikit-learn
NLTK
Streamlit
Pickle
